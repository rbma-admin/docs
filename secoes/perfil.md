# Meu perfil

Esta seção explica como qualquer pessoa logada (inclusive o administrador) edita
o próprio perfil: foto, dados pessoais e profissionais, biografia, visibilidade
na Rede de Apoiadores e links de redes sociais.

## Para que serve esta tela

O **Perfil** (menu lateral **Perfil**, endereço `/dashboard/perfil`) mostra seus
dados em modo de leitura. Para alterar qualquer coisa, clique em **Editar
perfil**, que abre o formulário em `/dashboard/perfil/editar`. Ao salvar, você
volta automaticamente para a visualização.

> Quem tem conta de **Pessoa Jurídica (PJ)** é levado para o perfil da
> organização (`/dashboard/empresa/perfil`), com campos próprios de empresa.
> As instruções abaixo descrevem o perfil de **Pessoa Física (PF)**.

## Editar o perfil — passo a passo

1. Em **Perfil**, clique em **Editar perfil**.
2. Altere os campos desejados nas seções descritas abaixo.
3. Clique em **Salvar** para gravar, ou **Cancelar** para descartar e voltar.
   Há também o link **Voltar para meu perfil** no topo.

### Seção "Foto e identificação"

- Aparece sua foto atual (ou as iniciais, se não houver foto).
- Botão **Alterar foto** — abre o seletor de arquivo. Aceita imagens JPEG, PNG
  ou WebP, com tamanho máximo de **5 MB**. Durante o envio o botão mostra
  "Enviando…".
- Botão **Remover foto** — aparece quando há uma foto; pede confirmação antes de
  apagar.

### Seção "Dados pessoais"

- **Nome** e **Sobrenome**.
- **Email** — exibido apenas para leitura (não é editável aqui).
- **Grau de instrução** — lista de opções.
- **Empresa atual** — texto livre.

### Seção "Biografia"

Campo de texto com formatação (negrito, itálico, listas etc.) para você se
apresentar.

### Seção "Rede de Apoiadores" (opção de aparecer na Rede)

> Esta seção só aparece para quem tem acesso à Rede de Apoiadores pelo plano. Sem
> esse acesso, marcar a opção não teria efeito, então ela fica oculta.

- Caixa de marcação **Aparecer na Rede de Apoiadores**.
- Quando marcada, seu perfil fica **visível** para outros apoiadores na página
  pública `/rede`. Você pode desmarcar a qualquer momento.

### Seção "Informações demográficas"

Campos opcionais: Identidade de gênero, Orientação sexual, Cor/Raça, População
tradicional e Informação de deficiência.

### Seção "Localização"

- **País** — lista de países.
- **Estado / província** — quando o país tem subdivisões cadastradas, vira uma
  lista; caso contrário, é um campo de texto. Só fica habilitado depois de
  escolher o país.

### Seção "Atuação profissional"

- **Área de atuação** — Academia, Terceiro setor, Governo, Consultoria ou Outra.
- **Cargo / profissão** — lista de opções.

### Seção "Experiência em monitoramento e avaliação"

- **Nível de experiência** — Iniciante, Intermediário ou Avançado.

### Seção "Networking"

- **Contato profissional** — texto livre (até 1000 caracteres) para indicar como
  outras pessoas podem entrar em contato com você profissionalmente.

### Seção "Redes sociais"

Permite cadastrar vários links profissionais (LinkedIn, Lattes, ResearchGate,
ORCID, GitHub, Twitter/X, Mastodon, Instagram — ou qualquer outro digitado à
mão).

1. Clique em **Adicionar rede**.
2. No campo de **plataforma**, digite o nome (há sugestões prontas) — até 60
   caracteres.
3. No campo de **URL**, cole o endereço completo (começando com `https://`). Se a
   URL for inválida, aparece o aviso "URL inválida. Use formato https://…".
4. Para remover um link, use o botão da lixeira ao lado dele.

Entradas totalmente vazias são descartadas ao salvar.

### Seção "Assinatura"

Mostra, **apenas para leitura**, o plano, o código, o status e a validade da sua
assinatura. Não é editável aqui.

## Pessoa Física x Pessoa Jurídica

- Conta **PF** vê todas as seções acima.
- Conta **PJ** é redirecionada para o **perfil da organização**, com campos de
  empresa em vez dos campos pessoais. As seções de biografia, demografia, Rede de
  Apoiadores e redes sociais pessoais não aparecem para conta PJ pura.
