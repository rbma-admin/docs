# Manual do Administrador — Sistema RBMA

Bem-vindo(a) ao manual de operação da plataforma **RBMA** (Rede Brasileira de Monitoramento e Avaliação).

Este guia é para o **administrador do sistema** — a pessoa que gerencia o conteúdo, os usuários e as
configurações pelo painel administrativo (`/dashboard`). Não é necessário conhecimento técnico.

## Como usar este manual

- Use o **menu à esquerda** para navegar pelas seções.
- Use a **busca** (no topo da barra lateral) para encontrar um assunto rapidamente.
- Cada seção descreve **o que a tela faz** e **o passo a passo** das ações mais comuns.

## Visão geral da plataforma

A RBMA tem duas partes:

- **Site público** — páginas abertas a qualquer visitante (início, sobre, notícias, formação,
  biblioteca pública, rede de apoiadores, contato) e o cadastro/login.
- **Painel administrativo** (`/dashboard`) — onde você, como administrador, gerencia tudo.

O acesso ao painel exige **login** e um **papel** (admin ou usuário). O administrador enxerga todas as
seções; um usuário comum vê apenas o próprio perfil e suas assinaturas.

## Primeiros passos

1. Acesse a página de **login** e entre com seu e-mail e senha de administrador.
2. Você cairá no **painel** (`/dashboard`). O menu lateral lista as seções que você pode gerenciar.
3. Comece pela seção desejada no menu à esquerda.
